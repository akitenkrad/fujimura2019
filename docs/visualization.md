# Visualization

[English] | [日本語](visualization.ja.md)

All tools resolve `results/latest` by default; pass `--results-dir` to target another run and `--output-dir` to redirect the PNGs.

## `fit-sem`

Estimates the ABM-induced SEM with [semopy](https://semopy.com/) from `agent_panel.csv`: builds one cross-sectional row per agent (time-averaged latent states + behaviour frequencies, excluding t=0), fits

```
fear ~ psafety ; acquiescent ~ fear ; voice ~ fear ; silence ~ acquiescent
```

and writes `sem_fit.json` with the standardized β̃, Wald 95% CIs, fit indices (CFI / GFI / RMSEA / χ²), `corr_silence_voice`, and the sign-match count against the paper anchors. Falls back to per-path OLS if semopy is unavailable.

## `visualize`

Reads `metrics.csv` (and `sem_fit.json` if present) and writes:

- `timeseries.png` — silence_rate / voice_volume / climate_of_silence over t, with the paper anchors (silence ≈ .38, voice ≈ .45) dotted.
- `motive_mix.png` — the 4-motive within-silent share trajectory.
- `path_diagram.png` — the SEM path diagram annotated with the estimated β̃ next to the paper values.

## `visualize-sweep`

Reads `sweep_summary.csv` and writes:

- `sweep_level_forest.png` — final silence_rate / voice_volume per hierarchy level L (mean ± SD), with the paper anchors.
- `sweep_climate_heatmap.png` — mean climate_of_silence over the (η × network_beta) grid — the structural conditions for the global silence spiral.

## `reproduce`

Runs `fit-sem` if needed, then writes `paper_fig1_path_diagram.png` (Fig.1-style) and `reproduction_report.json`, and prints the B1–B5 reconciliation table (sign match + CI overlap + the H5 silence ⊥ voice check).

## `show-experiment-settings`

Prints the run / sweep configuration and the `llm_meta.json` summary. Add `--json` for machine-readable output.
