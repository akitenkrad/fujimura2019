# CLI reference

[English] | [日本語](cli.ja.md)

The `fujimura` binary has four subcommands. The decision mode is a mutually-exclusive switch: `--decision-mode rule` (logistic ablation, no LLM) or `--decision-mode llm` (`socsim-llm`, Ollama-first → OpenAI fallback).

## `run`

Single configuration. Pools `--runs` seeds into one `agent_panel.csv` / `metrics.csv`.

| Flag | Default | Meaning |
|------|---------|---------|
| `--decision-mode` | `rule` | `rule` \| `llm` |
| `--locale` | `ja-JP` | `ja-JP` (L=4, ῑ=.55) \| `en-US` (L=3, ῑ=.40) |
| `--n-teams` | 5 | number of teams |
| `--team-size` | 80 | employees per team |
| `--n-levels` | locale default | hierarchy strength L (overrides locale) |
| `--eta` | 0.7 | supervisor-signal homogeneity η |
| `--network-beta` | 0.05 | Watts–Strogatz rewiring β |
| `--network-k` | 6 | Watts–Strogatz mean degree k |
| `--ivt-strength-mean` | locale default | initial IVT-strength mean |
| `--psafety-mean` | 0.613 | initial psychological-safety mean |
| `--prompt-variant` | `A` | prompt wording A / B / C |
| `--p-retaliate` | 0.05 | per-agent per-step retaliation probability |
| `--shock-t` | none | optional σ-shock step |
| `--t-max` | 12 | horizon |
| `--runs` | 1 | independent seeds (pooled) |
| `--seed` | 2019 | root seed |
| `--llm-temperature` | 0.0 | LLM temperature |
| `--llm-seed` | 0 | LLM seed offset |
| `--llm-model` | `llama3.1` | advisory model hint |
| `--cache-path` | `.llm_cache/cache.json` | prompt→response cache (LLM mode) |
| `--output-dir` | `results` | output base |

## `sweep`

Cartesian product over `--n-levels-values × (η range) × --network-beta-values × seeds`, one row per cell in `sweep_summary.csv`.

Key flags: `--n-levels-values 2,3,4,5`, `--eta-min/--eta-max/--eta-step`, `--network-beta-values 0.05,0.10,0.20`, `--runs`, `--t-max`, `--seed`.

## `cultural-compare`

Runs the JP and EN locales side by side (each with its locale defaults) into one pooled `agent_panel.csv`, for the cultural-ablation comparison of the path coefficients.

Key flags: `--decision-mode`, `--n-teams`, `--team-size`, `--eta`, `--t-max`, `--runs`, `--seed`, `--cache-path`.

## `reproduce`

Prints the entry points for the Python `fit-sem` / `reproduce` tooling that estimates the four path β̃ and reconciles them with the §5 paper anchors.
