# Architecture

[English] | [日本語](architecture.ja.md)

The simulation translates Fujimura & Hino's (2019) structural-equation model into a socsim `WorldState` advanced by 8 `Mechanism`s across socsim's fixed 6-phase loop.

## `SilenceWorld`

Employees live on a Watts–Strogatz organisational network (`socsim-net::SocialNetwork`); there is no spatial grid.

```
SilenceWorld {
  clock, employees: BTreeMap<AgentId, Employee>, teams: Vec<Team>,
  network: SocialNetwork, issue_salience σ(t), climate_of_silence C(t),
  locale: {JaJp | EnUs}, hierarchy_strength L, eta η, retaliation_this_step,
}
Employee {
  level, team, tenure, psych_safety ψ, fear f (Quiescent), acquiescent a (黙従),
  ivt_strength ι, expression, silence_motive: Option<Motive>, voice_threshold θ,
  voiced, silenced,   // two independent behavioural channels (paper H5)
}
Motive ∈ { Acquiescent, Quiescent, Prosocial, Opportunistic }
```

`BTreeMap` keys give a deterministic, sorted activation order.

### Independent voice / silence channels

The paper finds silence ⊥ voice (r = .02): they are *separate* behavioural dimensions, not opposites. The model therefore records two independent per-step flags — `voiced` (set by `voice_decision`) and `silenced` (set by `silence_spiral`) — rather than a single mutually-exclusive expression. `expression` is a display summary (Voice if voiced, else Silence if silenced, else Neutral); the panel CSV records the two flags separately.

## Initial distributions (paper Table 1, rescaled to [0,1])

Latent states are sampled IID from clipped Normals: ψ N(.613, .139), fear N(.450, .214), acquiescent N(.457, .197), ι N(.55 JP / .40 EN, .15), θ N(.40, .15).

## 8 mechanisms × 6 phases

| # | Mechanism | Phase | SEM path |
|---|-----------|-------|----------|
| 1 | `issue_salience` | Environment | σ(t) mean-reversion + optional shock |
| 2 | `retaliation_event` | Environment | probabilistic retaliation → `retaliation_this_step` |
| 3 | `fear_appraisal` | Decision | **ψ → fear** (paper β = −.54): `f ← f + α_f(g_f − f)`, `g_f` rises as ψ falls |
| 4 | `voice_decision[_rule]` | Decision | **fear → voice** (β = −.23): LLM (JP/EN prompt) or logistic ablation; sets `voiced` |
| 5 | `acquiescent_update` | Interaction | **fear → acquiescent** (β = +.86): `a ← (1-α_a)a + α_a σ_sat(γ_fa·f + ι)` |
| 6 | `silence_spiral` | Interaction | **acquiescent → silence** (β = +.52): `P(silence) = σ(β_a·a + β_C·(ρ−anchor) − β_θ·protect)`; sets `silenced` + motive |
| 7 | `psafety_update` | PostStep | voicing without retaliation slowly raises ψ |
| 8 | `climate_silence` | PostStep | aggregate C(t) for the next step |

All mechanisms use synchronous updates (snapshot → batch write). The LLM call is confined to `voice_decision`; everything else is deterministic.

## RNG streams

`derive_seed(root, &[label])` derives independent streams: `0` world-init (latent-state sampling), `1` engine (scheduler / Bernoulli draws), `2` network generation, `3` LLM `(agent_id, t)` seed. Rule mode is bit-deterministic; LLM mode is pseudo-deterministic via the prompt cache.

## β̃ estimation

The four SEM path coefficients are **not** hardcoded in Rust. The Python `fit-sem` tool reads `agent_panel.csv`, builds one cross-sectional row per agent (time-averaged latent states + behaviour frequencies, excluding t=0), and fits the SEM with semopy, reporting standardized β̃, Wald CIs, and the CFI/GFI/RMSEA fit indices.
